ChatGPT Translator 是一个开源的跨平台桌面客户端，使用 ChatGPT 来翻译文章和句子。

## 主要功能

-   无需选择源语言，由 ChatGPT 自动判断
-   支持多种语言互译
-   支持自定义快捷键
-   支持 macOS、Windows、Linux 平台

## 安装

1. Clone 项目到本地

```
git clone https://github.com/simpleapples/chatgpt-translator.git
```

2. 进入项目文件夹

```
cd chatgpt-translator
```

3. 安装依赖

```
npm i
```

4. 编译

```
npm run build
npm run package
```
